---
title: "Azure Form Recognizer"
publishedAt: "2022-01-04"
summary: "Form Recognizer is an Azure Applied AI service that extracts and analyzes form fields, text, and tables from documents using machine learning."
---

Form Recognizer is a cloud-based Azure Applied AI service that uses prebuilt and custom machine learning models to understand documents. It can extract structured data like fields and tables, detect text, and help automate document processing pipelines.
